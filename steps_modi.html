Using this voucher is very simple. You just need to follow the usual booking process on www.bookmyshow.com, as described below:
Log on to www.bookmyshow.com
Select Location, Movie, Cinema, Show Date, Time and Seats of your preference (Individual ticket price should be less than Rs.350 each. Maximum of two tickets can be booked using the voucher.)
On the payment page, tick the check box "Avail Discounts & Offers" and from the drop-down list, select the "Movie Money" option
Enter the Movie Money voucher code given in the blue box above and click on "Apply Offer". A Movie Money Voucher Code can be redeemed only once in a single transaction.
